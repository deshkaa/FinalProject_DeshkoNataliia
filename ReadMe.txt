Project name: 5 ways to explore San Francisco
Project idea: a user of this app will learn more about San Francisco most significant places. 
Android API to be used: 15 and newer.
Devices: smartphones.
Localozation: only English.

Project developer: Nataliia Deshko.

User can get to know about current time, weather and watch live cameras located in the city.
It's possible to look through information about each of places after horizontal scrolling of images and pressing on the corresponding one or its caption.
In all of the screens pictures and maps are provided. 
Also user can request more information about sights by sending email (after clicking "I want to get more information!" and entering email).
